# OpenClawSkills

Aletheia-grounded augmentation toolkit for AI orchestrators.

Compensates for model weaknesses with ontological middleware skills — turning less-capable orchestrator models into reliable operators through structured perception, memory, validation, planning, and safety.

## Quick Start

```bash
pip install openclaw-skills
openclaw-skills list
openclaw-skills config --config configs/default.yaml
```
